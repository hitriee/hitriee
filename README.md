<br>

### 1. Mental Assistant (멘털 어시스턴트) - AI를 활용한 감정 분석 서비스
- 역할 : 프론트엔드 / 디자인 제외한 화면과 기능 구현
- 사용 기술 : React, Typescript, Chakra UI 등
- 주요 성과
  - 하단 바, 하단 시트, 달력 등을 활용한 모바일 최적화 화면 구현
  - 기종별 UI 차이를 줄이기 위한 3개 이상의 기기 테스트
  - 기능 고도화의 일종으로 다국어 설정 기능 구현

<br>

### 2. PT하자 - AI를 활용한 발표 보조 플랫폼
- 역할 : 프론트엔드 / 퍼블리싱 이후 작업 수행, MVP 제작
- 사용 기술 : Next.js, Typescript 등
- 주요 성과
  - 사용자 편의성과 개발자 편의성을 고려한 데이터 형식 설계
  - CSS flex를 활용해 발표 대본과 실제 음성 비교 UI 구현
  - 스스로 목표를 설정하고 세부화하며 시간 관리 역량 향상
  - 웹소켓 통신 적용으로 기존 대비 발표 대본 조회 api 통신 시간을 약 20% 감축
